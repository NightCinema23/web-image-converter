# WebP to Image

> Microsoft Edge / Chrome 에서 WebP 이미지를 우클릭 한 번으로 **PNG** 또는 **JPG**로 변환 저장하는 브라우저 확장 프로그램

---

## 어떤 프로그램인가요?

웹에서 이미지를 저장하면 `.webp` 형식으로 저장되는 경우가 많습니다.  
WebP는 브라우저에서는 잘 보이지만, 카카오톡·파워포인트·포토샵 등 일반 프로그램에서 열리지 않는 경우가 있습니다.

이 확장 프로그램을 설치하면 이미지를 우클릭했을 때 **"PNG로 저장"** / **"JPG로 저장"** 메뉴가 생깁니다.  
클릭 한 번으로 변환과 저장이 동시에 완료됩니다.

- 별도 프로그램 설치 없음
- 인터넷 전송 없음 — 변환이 내 PC 브라우저 안에서만 처리됨
- Edge, Chrome 모두 사용 가능

---

## 설치 방법

### 1단계 — 파일 받기

페이지 오른쪽 위 **`<> Code`** 버튼 → **`Download ZIP`** 클릭

![Download ZIP 버튼 위치](https://docs.github.com/assets/cb-20363/mw-1440/images/help/repository/code-button.webp)

다운로드된 ZIP 파일을 **압축 해제**합니다.  
압축을 풀면 `web-to-image-main` 같은 폴더가 생깁니다. 이 폴더를 기억해두세요.

---

### 2단계 — 개발자 모드 켜기

1. Edge(또는 Chrome) 주소창에 아래를 입력하고 **Enter**

   - Edge: `edge://extensions`
   - Chrome: `chrome://extensions`

2. 열린 페이지 **왼쪽 아래**에서 **개발자 모드** 스위치를 켭니다

   | 끄기 | 켜기 |
   |------|------|
   | 회색 | 파란색 ✅ |

---

### 3단계 — 확장 프로그램 로드

1. **"압축 해제된 확장 프로그램 로드"** 버튼 클릭
2. 1단계에서 압축 해제한 폴더를 선택

   > ⚠️ ZIP을 풀면 폴더가 두 번 중첩되는 경우가 있습니다.  
   > `manifest.json` 파일이 들어 있는 폴더를 선택해야 합니다.

3. 목록에 **WebP to Image** 가 나타나면 설치 완료입니다 ✅

---

## 사용 방법

### 이미지 저장

저장하고 싶은 이미지 위에서 **마우스 오른쪽 버튼**을 클릭하면 메뉴가 나타납니다.

```
🖱️ 우클릭
 ├─ PNG로 저장   ← 투명 배경 유지, 고화질
 └─ JPG로 저장   ← 파일 크기 작음
```

저장된 파일명은 원본에서 확장자만 바뀝니다.
```
예시:  photo.webp  →  photo.png
       banner.webp →  banner.jpg
```

---

### 설정 변경 (JPG 품질 조절)

Edge 오른쪽 위 퍼즐 아이콘(🧩) → **WebP to Image** 클릭

| 설정 | 설명 |
|------|------|
| **PNG** | 투명 배경 유지, 파일 크기 큼 |
| **JPG** | 투명 배경 흰색 처리, 파일 크기 작음 |
| **JPG 품질** | 슬라이더로 50~100% 조절 (기본 92%) |

---

## PNG vs JPG 어떤 걸 써야 하나요?

| 상황 | 추천 |
|------|------|
| 로고, 아이콘 (투명 배경) | **PNG** |
| 사진, 풍경 이미지 | **JPG** |
| 파일 크기를 줄이고 싶을 때 | **JPG** |
| 화질 손실 없이 보관 | **PNG** |
| 모르겠으면 | **PNG** |

## 자주 묻는 질문

**Q. 우클릭 메뉴에 항목이 보이지 않아요.**  
A. 이미지 위에서 정확히 우클릭했는지 확인하세요. 빈 공간을 클릭하면 메뉴가 나오지 않습니다.

**Q. 클릭했는데 다운로드가 안 돼요.**  
A. 일부 사이트는 이미지 직접 접근을 차단합니다. 해당 사이트에서는 동작하지 않을 수 있습니다.

**Q. Chrome에서도 쓸 수 있나요?**  
A. 네. `chrome://extensions` 접속 후 동일한 방법으로 설치할 수 있습니다.

**Q. 개인 정보가 유출되지 않나요?**  
A. 변환 작업이 100% 브라우저 내부에서 처리됩니다. 이미지 데이터가 외부 서버로 전송되지 않습니다.

---

## 제거 방법

`edge://extensions` 접속 → **WebP to Image** 카드에서 **"제거"** 클릭

## 기술 정보

| 항목 | 내용 |
|------|------|
| 플랫폼 | Manifest V3 (Chrome / Edge) |
| 이미지 변환 | 브라우저 내장 OffscreenCanvas API |
| 외부 통신 | 없음 |
| 필요 권한 | `downloads`, `contextMenus`, `storage`, `<all_urls>` |
